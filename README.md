# Pidro

A card game you've probably never heard of. Unless you're from Vasa Finland or Louisiana. Then you definitely have.

Building this with good ol' [Ralph](https://ghuntley.com/ralph/). Turns out pair programming with an AI is basically having a co-founder who never asks for equity.

Phoenix + Elixir because when you need to handle a bunch of people playing cards at the same time, might as well use the best hammer for that kind of thing.
